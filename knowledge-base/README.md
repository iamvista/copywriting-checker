# Vista 知識庫

這個資料夾用來存放 Vista 的著作、文章和課程資料，作為文案健檢工具的分析依據。

## 📂 資料夾結構

```
knowledge-base/
├── books/          # 您的書籍（PDF、Word 等）
├── articles/       # 文章、電子報（可存文字檔或連結清單）
└── courses/        # 課程講義、簡報
```

## 📥 如何新增資料

### 方式 1：直接拖曳檔案
1. 在 Finder 中打開這個資料夾
2. 把 PDF、Word、PPT 等檔案拖進對應的子資料夾
3. 告訴 Claude 檔案名稱即可

### 方式 2：提供網路連結
直接在對話中給 Claude URL：
- 部落格文章
- Medium
- 電子報存檔
- 網站頁面

### 方式 3：文字檔
如果有純文字內容，可以存成 .txt 或 .md 檔案放進來
